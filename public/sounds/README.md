# 🔊 Звуковые файлы для игры "100 к 1"

## 📁 Необходимые звуки:

Поместите следующие звуковые файлы в эту папку:

### Основные звуки игры:

1. **correct.mp3** - Правильный ответ (открытие карточки)
   - Описание: Восходящий приятный тон
   - Длительность: ~0.3-0.5 сек
   - Пример: "дзинь!" или восходящий ксилофон

2. **wrong.mp3** - Неправильный ответ (ошибка)
   - Описание: Низкий гудок или зуммер
   - Длительность: ~0.5-1 сек
   - Пример: как звук "Х" в оригинальной игре

3. **flip.mp3** - Переворачивание карточки
   - Описание: Короткий щелчок
   - Длительность: ~0.1 сек
   - Пример: "щелк" или "клик"

4. **round-start.mp3** - Начало раунда
   - Описание: Фанфары или торжественная музыка
   - Длительность: ~2-3 сек
   - Пример: музыкальная заставка раунда

5. **victory.mp3** - Победа
   - Описание: Радостная мелодия
   - Длительность: ~3-5 сек
   - Пример: фанфары победы

6. **tick.mp3** - Тик таймера (для финала)
   - Описание: Короткий тикающий звук
   - Длительность: ~0.05 сек
   - Пример: "тик" часов

7. **final-music.mp3** - Музыка финального раунда
   - Описание: Напряженная фоновая музыка
   - Длительность: ~30-60 сек (зацикливается)
   - Пример: драматичная музыка

8. **drumroll.mp3** - Барабанная дробь
   - Описание: Для раскрытия результатов
   - Длительность: ~2-3 сек
   - Пример: барабанная дробь

## 🎵 Где найти звуки:

### Вариант 1: Бесплатные библиотеки звуков
- **Freesound.org** - https://freesound.org/
- **Zapsplat** - https://www.zapsplat.com/
- **Mixkit** - https://mixkit.co/free-sound-effects/
- **BBC Sound Effects** - https://sound-effects.bbcrewind.co.uk/

### Вариант 2: YouTube Audio Library
- https://www.youtube.com/audiolibrary

### Вариант 3: Создать самому
- **Audacity** (бесплатно) - https://www.audacityteam.org/
- **GarageBand** (Mac, бесплатно)
- **FL Studio** (платно)

## 🔧 Требования к файлам:

- **Формат:** MP3 или WAV
- **Битрейт:** 128-192 kbps (для MP3)
- **Размер:** Желательно < 500KB каждый
- **Моно/Стерео:** Стерео предпочтительнее
