# 🎮 100 к 1 - Интерактивная игра

Современное веб-приложение для популярной телевизионной игры "100 к 1" (Family Feud) с системой раундов и захватывающими визуальными эффектами!

## 🚀 Технологии

- **React 18** - современная библиотека для создания пользовательских интерфейсов
- **Vite** - быстрый сборщик и dev-сервер
- **Tailwind CSS** - utility-first CSS фреймворк
- **Framer Motion** - мощная библиотека для анимаций
- **Lucide React** - красивые иконки

## ✨ Возможности

### Игровые механики
- 👥 **Игра для двух команд** - настоящее соревнование как в ТВ шоу!
- 🔄 **Система ходов** - команды играют по очереди
- ⚡ **Механика отбоя** - если команда получила 3 ошибки, соперник может отобрать очки!
- 🎯 **Система раундов** - 4 различных типа раундов основной игры:
  - **Простой раунд** (×1) - обычные очки
  - **Двойной раунд** (×2) - все очки удваиваются!
  - **Тройной раунд** (×3) - все очки утраиваются!
  - **Обратный раунд** - наименее популярные ответы дают больше очков!
- 🎲 **20 уникальных вопросов** для основной игры + **5 вопросов** для финала
- 👑 **Финальный раунд "Большая игра"** - два участника команды-победителя пытаются набрать 200 очков!
- ⏱️ **Таймер в финале** - по 20 секунд на каждый вопрос
- 📊 **Раздельный подсчёт очков** - каждая команда накапливает свои очки
- ❌ **Отслеживание ошибок** - у каждой команды по 3 попытки на раунд

## 🎮 Как играть

### Подготовка:
1. **Введите названия команд** - придумайте название для двух команд
2. **Нажмите "Начать игру"** - игра начнётся с заставки раунда

### Основная игра (20 вопросов):
1. **Командная игра** - две команды соревнуются друг с другом
2. **Ходы команд** - команды по очереди отвечают на вопросы
3. **Открывайте ответы** - кликайте на синие карточки с номерами
4. **Следите за ошибками** - у каждой команды есть 3 попытки на ошибку
5. **Система отбоя** ⚡:
   - Команда получила 3 ошибки? Соперник может попробовать отобрать очки!
   - Если соперник угадает хотя бы один ответ - он забирает все очки
   - Если соперник тоже ошибётся - очки остаются у первой команды
6. **Управление ходом**:
   - 🔴 "Неправильный ответ" - добавить ошибку команде
   - 🔄 "Передать ход" - передать ход другой команде вручную

### Финальный раунд "Большая игра" 🏆:
После завершения 20 вопросов команда-победитель может сыграть в финал!

1. **Два участника** - из команды-победителя выбираются 2 игрока
2. **Игрок 1** отвечает на 5 вопросов:
   - На каждый вопрос 20 секунд
   - Нужно быстро ответить первое, что приходит в голову
3. **Игрок 2** отвечает на те же вопросы:
   - Не видел ответов первого игрока
   - Тоже по 20 секунд на вопрос
4. **Цель** - набрать вместе 200 очков!
5. **Результат**:
   - ✅ 200+ очков = ПОБЕДА в большой игре! 🎉
   - ❌ Меньше 200 = Попытка не удалась, но основная победа остаётся!

### Раунды основной игры:
- **Простой раунд** 🏆 - классическая игра, обычные очки (5 вопросов)
- **Двойной раунд** ⚡ - все очки ×2! (5 вопросов)
- **Тройной раунд** 🔥 - все очки ×3! (5 вопросов)
- **Обратный раунд** 🔄 - самые редкие ответы дают больше очков! (5 вопросов)
- **Финальный раунд** 👑 - "Большая игра" для команды-победителя (5 вопросов на скорость)

## 🎯 Ключевые фичи кода

- **Командная система** - полноценная логика для двух команд с раздельным подсчётом
- **Механика отбоя** - система кражи очков при 3 ошибках команды
- **Управление ходами** - отслеживание активной команды и переключение ходов
- **Финальный раунд** - отдельный режим игры с таймером и двумя участниками
- **Система таймера** - обратный отсчёт 20 секунд на каждый вопрос в финале
- **Автоматическая проверка ответов** - сопоставление ответов игроков с правильными
- **Framer Motion** - все анимации реализованы через декларативный API
- **React Hooks** - современное управление состоянием (useState, useEffect)
- **Система множителей** - динамический расчёт очков в зависимости от раунда
- **Обратный раунд** - специальная логика инверсии очков
- **Переходы раундов** - автоматическое отображение заставок при смене раундов

## 🎮 Игровые режимы
